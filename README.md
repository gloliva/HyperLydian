HyperLydian
===========

## Install
```
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

## Run Game
```
python3 game/main.py
```

## Controls
Subject to change (definitely will change)

`Arrow Keys` - Move

`W` - Shoot

`Q` - Rotate CCW

`E` - Rotate CW
