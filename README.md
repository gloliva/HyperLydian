HyperLydian
===========

## Install
```
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

## Run Game
```
python3 game/main.py
```

## Where things are at
No GUI yet, but it will happen.

Extremely in-progress; everything subject to change.

## Controls

`Arrow Keys` - Move

`W` - Shoot

`R` - Cycle Weapons

`Q` - Rotate CCW

`E` - Rotate CW
